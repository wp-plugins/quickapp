== Installation ==
= Installing the Quickapp WordPress mobile App Plugin from your WordPress admin panel: =
<p>
 <ol>
  <li>Go to the "Plugins" / "Add new" on the left main menu.</li>
  <li>Type Quickapp within the search box and click "Search Plugins".</li>
  <li>Choose the "Quickapp – Free Mobile App for your WP blog (Android, iOS, HTML5)" Plugin and click "Install Now".</li>
  <li>Click "Activate Plugin".</li>
  <li>Hover on Quickapp icon on the left main menu and click on setting link to start creating your own mobile App.</li>
 </ol>
</p>
= Download and Install the Quickapp mobile App Plugin from wordpress plugin directory or from the Quickapp website: =
<p>
 <ol>
  <li>Click the "Download" button.</li>
  <li>Go to your WordPress admin panel, open the Plugins menu on the left main menu and click "Add New".</li>
  <li>Click upload on the top page menu.</li>
  <li>Choose the "Quickapp – Free Mobile App for your WP blog (Android, iOS, windows and HTML5)" plugin and click "Install Now".</li>
  <li>Click "Activate Plugin".</li>
  <li>Click the Quickapp button on the left main menu to start creating your own mobile App.</li>
 </ol>
</p>

== Frequently Asked Questions ==
= What is Quickapp? =
Quickapp is a wordpress plugin that converts your blog or website into an HTML5 mobile app for all smartphones platforms including android, iphone and Windows.

= Who can create a mobile App using the Quickapp Plugin? =
Any WordPress powered website that has the capability to install a WordPress plugin can use Quickapp and create his own mobile App

= What version of WordPress do I need? = 
The Quickapp plugin is compatible with WordPress versions 3.0.1 and up.

= How do I install APK file to my Android without Google Play? =
Manual installation of apps using the .apk files, you need to allow your phone to install from "Unknown Sources" (i.e. non-Market apps).
<ul>
 <li>Navigate to Menu -> Settings -> Applications or Menu -> Settings -> Security and check the box marked "Unknown Sources".</li>
 <li>Android does not natively come with any method of browsing the data on your SD card, so you will need to install a file manager from the market. (i.e. ASTRO File Manager, ES File Explorer)</li>
 <li>Download .apk file to SD card.</li>
 <li>On your Android device, navigate to the .apk file using Manager Manager and select it. This will open a dialog box allowing you to install the app. Select "Open App Manager".</li>
 <li>On the next two pages, select "Install" and "Install" again to install the .apk.</li>
 <li>Your new app is now installed.</li>
</ul>
